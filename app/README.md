# Hello World eksempel med PhoneGap

## PhoneGap Guide med hello world eksempel

1. Download og installer PhoneGap: http://docs.phonegap.com/getting-started/1-install-phonegap/desktop/
2. Når PhoneGap er installeret vælger du open existing project og vælger dette projekt
3. Efterfølgende vil der stå server is running on "http.//"server adressen", klik på den og du vil få hello world eksemplet frem.

### PhoneGap Mobile View (testet på Android - Google Play)

1. Download PhoneGap Developer fra Google Play
2. Indtast server linket og du vil få vist eksemplet frem
